Detailed Description

Assign wrenches to segment based on a given mapping method.

The mapping method is used to determine which force plateform is mapped whith which segment. If several wrenches are assigned to the same segment, they are averaged.

The assigned wrench is set into the distal joint wrench of the selected segment.
