Detailed Description

Manual mapper between force platform and segment to be used with the class btk::ModelWrenchAssembly.

To map manually a wrench with a segment, you have only to use the method AppendManualRelation(). This only take the index and the definition (type, side) of the segment. Several force platforms can be mapped to the same segment. For example, the following code add a relation between the force platforms #0, #1 and the left foot segment:

SegmentForcePlatformWrenchManualMapper::Pointer mapper = SegmentForcePlatformWrenchManualMapper::New();
mapper->AppendManualRelation(0,Segment::Foot,Segment::Left);
mapper->AppendManualRelation(1,Segment::Foot,Segment::Left);

To update a model with this mapping, you need to use the class btk::ModelWrenchAssembly.
