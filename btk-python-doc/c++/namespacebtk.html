btk::RawFileStream

Raw file stream used by the btk::BinaryFileStream class. Depending your configuration, this raw stream will correspond to btk::mmfstream (default) or std::fstream. The class btk::mmfstream uses the memory mapped file mechanism.

Warning
It is not adviced to used this class directly. Even if you can potentially speedup data's reading by accessing directly to the buffer, it is also a large source of error and crash (buffer overlfow, etc.)

BTK_MODELS_EXPORT void btk::ExtractRequiredPoints ( std::map< std::string, Point::Pointer > *  landmarks,
Acquisition::Pointer  input,
LandmarksTranslator::Pointer  lt,
const std::vector< std::string > &  ll 
)

Try find the required points and set them in landmarks. If a landmark translator is used, then this method try first to use it. If no point is found with the translation, then then the original label for the landmarks (set in ll) is used. In both cases (point found with translated/original label), residuals are checked is the point is valid. By valid it means that at least one frame has a positive or null residual.
