Detailed Description

Add BSPs to a given model using the alternative data published by de Leva (1996).

Note: The CoM of the foot is adapted in taking account the fact that the origin proposed by the author is the heel while BTK use the ankle joint center.

Reference
de Leva P.
Adjustments to Zatsiorsky-Seluyanov’s segment inertia parameters
Journal of Biomechanics, 1996, 29(9), 1223-1230.
