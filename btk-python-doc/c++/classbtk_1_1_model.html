void btk::Model::Copy ( Model::Pointer  toCopy,
int  option = All 
)

Clear the model (this) and deeply copy the informations of the given model toCopy (label, description, parameter) depending of the given option. The available options are defined by the following values:

  • Model::CopyOption::All: (default) Deep copy of all the data including the joint wrenches and the motion of the segment.
  • Model::CopyOption::OnlyDefinition: Deep copy of all the data but the data related to the motion and the joint wrenches are reset and the frame number is set to 1.
  • Model::CopyOption::OnlyJointWrenches: Deep copy of the data and the joint wrenches, but the motion of the segment is set by a shallow copy.

The option OnlyDefinition and OnlyJointWrenches can be used to reduce the memory consumption of the object if in some case, you need only the definition of the model or you want to modify only the joint wrenches.

Note: The configurator is every time assigned by a shallow copy.
