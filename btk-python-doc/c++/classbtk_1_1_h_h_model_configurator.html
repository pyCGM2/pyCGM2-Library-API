Detailed Description

Configurator for the Helen Hayes marker set.

Not possible to use the parameters 'UseNodeLHipCenterAsLeftHJC' and 'UseNodeRHipCenterAsRightHJC' due to the way this model is implemented

Reference
Kadaba M. P., Ramakrishnan H.K., Wootten M.E., Gainey J., Gorton G., Cochran G.V.
Repeatability of kinematic, kinetic, and electromyographic data in normal adult gait. J Orthop Res, 7 (6), 1989, 849-860.

void btk::HHModelConfigurator::Configure ( Model *  model,
BodyPart  b,
Side  s 
) const
virtual

Configure the model by defining these segments and joints. The generic configuration is defined with:

  • 2 segments by upper limb (arm and foream);
  • 3 segments by lower limb (thigh, shank and foot);
  • 1 head segment
  • 1 torso segment;
  • 1 pelvis segment;
  • 2 relative joints by upper limb (shoulder, elbow);
  • 3 relative joints by lower limb (hip, knee and ankle);
  • 1 global joint for the head;
  • 1 global joint for the torso;
  • 1 global joint for the pelvis;
