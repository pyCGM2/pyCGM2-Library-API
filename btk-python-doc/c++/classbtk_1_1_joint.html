Detailed Description

Class to define a joint, defined by a proximal and distal segment.

A joint is supposed to have 6 DOFs, so without any constraint. To define a joint between a segment and the global frame, you have only to define global frame as an empty segment (i.e. use btk::Segment::Pointer()). For example, to define the joint "thorax" between the global frame and the torso you can use the following code:

btk::Segment::Pointer torso = btk::Segment::New(Segment::Torso, Segment::Central, "Upper part of the trunk", "");
btk::Joint::Pointer thorax = btk::Joint::New(btk::Segment::Pointer(), torso,"Thorax", "Joint between the global frame and the trunk")
