Detailed Description

Motion builder based on the point sets fiting method of Horn (1987).

The fitting is realized with the markers registered in each segment with the attribute btk::Segment::SCSBuilder and existing in the given input acquisition. A minimum of 3 markers is required to determine a segment's pose.

Reference
Horn B.K.P.
Closed-form solution of absolute orientation using unit quaternions
Journal of the Optical Society of America A, 1987, 4(4), 629-642.
