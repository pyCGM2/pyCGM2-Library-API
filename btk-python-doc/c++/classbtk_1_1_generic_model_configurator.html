Detailed Description

Generic configurator.

This class is dedicated to be inherited as it only configures the model and do not do any computation. Moroever, this configurator creates constraints on joint wrenches to have for most of the joint:

  • the distal joint wrench set as the reaction of the proximal joint wrench of the distal segment ;
  • the position of the proximal joint wrench is set to the origin of the segment.

This last constraint implies that the origin of the constrained segment MUST be set to its proximal end point.

The full detail of this configuration are presented in the description of the method GenericModelConfigurator::Configure().

void btk::GenericModelConfigurator::Configure ( Model *  model,
BodyPart  b,
Side  s 
) const
virtual

Configure the model by defining these segments and joints. The generic configuration is defined with:

  • 3 segments by upper limb (arm, foream and hand);
  • 3 segments by lower limb (thigh, shank and foot);
  • 1 head segment
  • 1 torso segment;
  • 1 pelvis segment;
  • 3 relative joints by upper limb (shoulder, elbow and wrist);
  • 3 relative joints by lower limb (hip, knee and ankle);
  • 1 relative joint between the head and the torso;
  • 1 relative joint between the torso and the pelvis;
  • 4 fixed chains:
    • Left/Right lower limbs: thigh, shank and foot
    • Left/Right upper limbs: arm, foream and hand

Each distal joint wrench is set as the reaction of the proximal wrench of the distal segment:

  • left forearm distal joint wrench set as the reaction of the left hand proximal joint wrench;
  • left arm distal joint wrench set as the reaction of the left forearm proximal joint wrench;
  • right forearm distal joint wrench set as the reaction of the right hand proximal joint wrench;
  • right arm distal joint wrench set as the reaction of the right forearm proximal joint wrench;
  • left shank distal joint wrench set as the reaction of the left foot proximal joint wrench;
  • left thigh distal joint wrench set as the reaction of the left shank proximal joint wrench;
  • right shank distal joint wrench set as the reaction of the right foot proximal joint wrench;
  • right thigh distal joint wrench set as the reaction of the right shank proximal joint wrench.

Moreover, the position of some proximal wrenches are constrained:

  • left hand proximal joint wrench position set from the origin of the left hand segmental frame;
  • right hand proximal joint wrench position set from the origin of the right hand segmental frame;
  • left forearm proximal joint wrench position set from the origin of the left forearm segmental frame;
  • right forearm proximal joint wrench position set from the origin of the right forearm segmental frame;
  • left arm proximal joint wrench position set from the origin of the left arm segmental frame;
  • right arm proximal joint wrench position set from the origin of the right arm segmental frame;
  • left foot proximal joint wrench position set from the origin of the left foot segmental frame;
  • right foot proximal joint wrench position set from the origin of the right foot segmental frame;
  • left shank proximal joint wrench position set from the origin of the left shank segmental frame;
  • right shank proximal joint wrench position set from the origin of the right shank segmental frame;
  • left thigh proximal joint wrench position set from the origin of the left thigh segmental frame;
  • right thigh proximal joint wrench position set from the origin of the right thigh segmental frame;
