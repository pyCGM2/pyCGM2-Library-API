Detailed Description

Container to provide the reaction of a wrench witout extra storage.

Shallow copy of a wrench to return the reaction force and moment when using methods GetForce() and GetMoment() without the need to store redudant information.

Warning
Cloning a WrenchReaction will return a Wrench object with a deep copy of the position, force and moment.
