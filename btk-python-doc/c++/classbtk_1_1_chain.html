Detailed Description

Abstract class for the definion a chain of segments.

The chain defined in each model will be used for example with the inverse dynamic. This latter use the chain to know from which segment to start the computation until which end.

When a chain is inserted in a model (

See Also
Model::AppendChain()), then this one is directly attached to this chain.
