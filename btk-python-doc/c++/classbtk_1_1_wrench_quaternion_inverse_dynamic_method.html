Detailed Description

Compute forces and moments from the wrench and quaternion inverse dynamic method.

Assumptions:

  • The origin for the intertial frame of each segment is set to the proximal endpoint of the segment

Note: This implementation does not require to set the groud reaction force to the endpoint (distal joint wrench) of the segment. Instead of this, you can set it as an external wrench (for example the Segment::ForcePlatformWrench or a Segment::UserWrench). With this implementation it gives the possibility to have several segments with external wrenches (other than the extremities). An exemple could be the use of a multi-segment foot model (Oxford foot model?) where the groud reaction wrenche are splitted into the segments.

Reference
Dumas R.; Aissaoui R.; de Guise J.A.
A 3D generic inverse dynamic method using wrench notation and quaternion algebra
Comput Methods Biomech Biomed Engin., 2004 Jun, 7(3), 159-66.

Method containing the algorithm to compute dynamic inverse.

The values of the array gravity must be set in meter by second squared (m/s²).
The parameter scaleToMeter is a scale factor used to compute the forces and the moment depending of the linear acceleration and the tensor of inertia. To be homogeneous in the units, the computation must use a mass set in kilogram (kg), an acceleration set in meter by second squared (m/s²) and a tensor of inertia in kg·m². So, if the unit of the trajectories are the millimeter, then the variable scaleToMeter must be set to 0.001 to scale correctly the internal variables.
